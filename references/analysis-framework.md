## 维度一：需求画像——从“表象”到“本质”

### 显性需求（表层动机）

客户直接陈述的购房理由。归类为：

| 需求类型 | 说明 |
|---------|------|
| 婚房置业 | 为结婚准备的住房 |
| 学区需求 | 子女教育配套需求 |
| 改善换房 | 提升居住品质 |
| 养老安居 | 为父母或自己养老 |
| 投资增值 | 资产保值增值 |
| 资产配置 | 财富多元化配置 |
| 暂无明确目的 | 闲看、了解市场 |

### 隐性需求（深层驱动）

结合客户背景（年龄、职业、籍贯、教育背景、财富层级）与言行，剖析真实动机。

#### 关键思考题

| 思考维度 | 核心问题 |
|---------|---------|
| **时机** | 为什么是现在看房？（触发事件） |
| **动机** | 为什么看我们楼盘？（吸引点） |
| **匹配** | 为什么关注这个户型/产品？（价值匹配） |
| **解码** | 某句特定的话背后可能的意思是什么？（如“再看看”可能是价格异议、决策权缺失或对比心理） |

## 维度二：产品反馈——关注点与抗性点

### 满意度分析

客户对以下方面明确表示认可的点：

| 类别 | 具体维度 |
|------|---------|
| **品牌价值** | 品牌实力、开发商信誉 |
| **地段价值** | 地段价值、区域发展 |
| **周边配套** | 商业、交通、医疗 |
| **社区规划** | 社区规划、园林景观 |
| **产品设计** | 户型设计、面积段、楼层视野 |
| **装修标准** | 装修标准、用材品质 |
| **公共区域** | 大堂、电梯、会所等 |
| **景观资源** | 公园、水景、城市景观 |

### 抗性点分析

客户明确表示不满、多次询问或犹豫的点。**关键区分**：

- **硬伤**：不可改变的因素（楼层、朝向、临路噪音）
- **可解释的误解**：对规划、政策的疑虑（可通过沟通化解）
- **可协调的障碍**：价格、付款方式（可通过商务政策解决）

## 维度三：购买逻辑——决策结构与客观条件

### 决策三角分析

| 角色 | 定义 | 关键问题 |
|------|------|---------|
| **决策人** | 拥有最终决定权的人 | 是本人、配偶还是父母？ |
| **出资人** | 资金主要来源 | 自有、父母资助还是多方凑集？ |
| **使用人** | 未来实际居住者 | 自己、子女还是父母？ |

**关键分析**：三者的意见是否统一？不统一时的矛盾点是什么？

### 客观条件扫描

| 维度 | 检查项 |
|------|-------|
| **购房资格** | 是否存在限购问题？是否有购房名额？ |
| **资金状况** | 首付资金是否已到位？是否需要资产处置（卖旧买新）？ |
| **决策障碍** | 关键决策人是否未到场？是否需要家人复看？ |

### 市场预期

| 分析维度 | 关键问题 |
|---------|---------|
| **市场认知** | 对经济形势和市场如何看待？认为房价涨跌趋势如何？ |
| **决策态度** | 是着急决定，还是观望等待？等待的点是什么？如何触发决策？ |
| **议价能力** | 后续议价空间如何判断？对价格的敏感度？ |

## 维度四：交易画像——房源、价格与周期

### 房源匹配

| 分析项 | 说明 |
|--------|------|
| **首选房源** | 客户明确意向的房源（楼栋、楼层、户型） |
| **备选房源** | 客户可接受的替代房源 |
| **在售状态** | 意向房源是否在售？有无其他客户竞争？ |
| **替代方案** | 若首选存在抗性，可引导的方向：<br>- 总价不变面积稍小<br>- 面积不变楼层稍低<br>- 接受更高总价获取更优资源 |

### 竞品对比

| 竞品类型 | 对比维度 |
|---------|---------|
| **新房竞品** | 品牌、区位、配套、户型、园林、质量、物业 |
| **二手房选项** | 区位、年限、品质、学区、装修、价格 |

### 价格与付款

| 维度 | 关键信息 |
|------|---------|
| **心理价位** | 客户透露或可推断的预算范围 |
| **价格弹性** | 是否有上探空间？对价格的敏感度如何？ |
| **付款方式** | 全款/贷款。若贷款：<br>- 首付比例<br>- 征信情况（需提醒销售关注）<br>- 低首付方案能否撬动更高预算 |

### 成交周期预测

| 维度 | 分析要点 |
|------|---------|
| **下次接触点** | 预计何时复访/下次通话？（本周末、下个月、待定） |
| **关键时间窗** | 近期是否有节日、家庭聚会等决策契机？ |
| **决策节奏** | 若成交，预计支付定金、签署合同、付清首付的时间预期？（关联促销政策：如7天内付首付享额外折扣） |

## 维度五：客户定性——意向度与策略锚定

### 意向度评级

| 等级 | 判定标准 |
|------|---------|
| **高** | 需求明确、决策人已到访、资金基本落实 |
| **中** | 有兴趣但有关键问题未决 |
| **低** | 闲看、对比初期、客观条件不成熟 |

### 成交概率评估

| 等级 | 判定标准 |
|------|---------|
| **高** | 房源价格匹配、决策链短 |
| **中** | 需解决1-2个核心问题 |
| **低** | 存在难以逾越的障碍 |

### 客户价值分类

| 类别 | 定义 | 跟进策略 |
|------|------|---------|
| **A类：快速成交型** | 条件成熟，需快速逼定 | 短期集中跟进，锁定房源与优惠 |
| **B类：重点培育型** | 有兴趣但有障碍待解决 | 针对性解决核心问题，设定跟进节奏 |
| **C类：长期观察型** | 条件暂不成熟 | 定期维系，等待时机 |

> **详细等级评定标准参见 `customer-rating.md`**
